# Rubinius::Webconsole

Web interface for Rubinius VM agent.

## Installation

Add this line to your application's Gemfile:

    gem 'rubinius-webconsole'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install rubinius-webconsole
